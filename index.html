Definisi HTML

HTML adalah bahasa standar untuk pembuatan halaman web

  • HTML kepanjangan dari Hyper Text Markup Language
  • HTML menggambarkan struktur dari halaman web menggunakan markup
  • Elemen HTML merupakan bagian pembangun dari halaman HTML
  • Elemen HTML diwakili dengan Tag
  • Tag HTML memiliki konten seperti "heading", "paragraph", "table" dan seterusnya
  • Tag HTML tidak ditampilkan pada Browser, hanya menampilkan konten halaman

Struktur HTML

Contoh Struktur HTML:

<!DOCTYPE html>
<html>
<head>
<title>
<body>

<h1> This is a Heading </h1>
<p> This is a Paragraph </p>
</body>

Penjelasan :

  • Tag <!DOCTYPE html> mendefinisikan dokumen ini menggunakan HTML 5
  • Tag <html> merupakan elemen utama dari halaman web
  • Tag <head> merupakan elemen yang mengandung informasi meta tentang dokumen halaman web
  • Tag <title> merupakan elemen yang menentukan judul dokumen halaman web
  • Tag <body> merupakan elemen yang berisi konten halaman web
  • Tag <h1> merupakan elemen yang mendefinisikan heading
  • Tag <p> merupakan elemen yang mendefinisikan paragrap
